# Projetos de Ciência de Dados

Esta pasta contém projetos de Ciência de Dados desenvolvidos de forma independente,
com foco em aprendizado contínuo, análise de dados, machine learning e aplicação
em cenários realistas de negócio.

Cada projeto possui:
- Contexto do problema
- Análise exploratória dos dados (EDA)
- Modelagem e avaliação
- Conclusões e próximos passos

Os datasets utilizados são públicos ou simulados para fins educacionais.
